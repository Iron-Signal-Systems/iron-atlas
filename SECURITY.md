# Security Policy

Iron Atlas is pre-alpha and is not ready for production use.

## Never Commit

- Raw firewall configuration backups
- Cisco technical-support reports
- Unredacted command transcripts
- Passwords, tokens, shared secrets, private keys, or certificate private material
- Production IP inventories that are not approved for repository storage
- NPS, TACACS+, SNMP, RADIUS, VPN, or API credentials

## Reporting

Do not open a public issue containing sensitive infrastructure details. Use a private, approved Iron Signal Systems reporting channel.

## Governing Principle

A successful parser or collection proves only that evidence was obtained and interpreted to the tested extent. It does not prove the device is healthy, secure, correctly configured, or synchronized with documentation.
