## Intended use

The logo may be used in:

- the Iron Atlas repository README;
- project documentation;
- project-owned diagrams;
- presentation and report covers;
- approved project websites;
- release artwork; and
- other Iron Signal Systems material that accurately represents Iron Atlas.

## Accessibility

Every rendered use must include meaningful alternative text. A recommended
description is:

> Iron Atlas shield emblem with compass, network-circuit elements, and the words
> “Network & Security Intelligence” and “Iron Signal Systems.”

The image must not be the sole source of product name, status, instructions, or
security meaning.

## Assurance boundary

The logo is a visual identity asset.

It is not:

- a cryptographic signature;
- a repository-validation result;
- an acceptance mark;
- a certification;
- proof of an official release;
- proof of production readiness; or
- proof that an artifact was published by Iron Signal Systems.

Authenticity and release status are established only through the applicable
signed source history, signed tags, release provenance, checksums, manifests,
and accepted project evidence.

## Modification

Material logo changes should:

1. preserve the original asset;
2. create a separately named candidate;
3. document the reason for the change;
4. verify accessibility and legibility;
5. update this record and the source manifest; and
6. receive normal project review before replacing the canonical asset.

Lossless optimization that does not change rendered pixels may replace the
stored bytes only through a reviewable change that records the old and new
digests.
