# Synthetic Representative-Provider Evidence Fixture

This directory contains a synthetic, sanitized, digest-bound example of the
representative-provider evidence contract.

It contains no real provider capture, JWT, token response, cookie, credential,
private key, subject, username, email address, client identifier, or production
configuration. Values under `provider.example.invalid` and
`urn:iron-atlas:test:*` are reserved synthetic fixtures.

The bundle is observation-only and explicitly makes no compatibility claim.
